# royal-hotel-website

# Royal Hotel Website

A responsive and modern hotel website developed using HTML, CSS, and JavaScript. This project showcases hotel information, room details, gallery images, and a booking/contact form.

## Live Features

- Responsive design for desktop, tablet, and mobile devices
- Hotel homepage with hero section
- About page with hotel information
- Rooms page with room categories and pricing
- Gallery page displaying hotel images
- Contact page with booking form
- JavaScript form validation
- SEO-friendly meta tags
- Modern user interface

# How To Run The Project

## Method 1: Live Server

1. Open project folder in VS Code
2. Install Live Server Extension
3. Right-click on index.html
4. Click "Open with Live Server"

## Method 2: Direct Browser

1. Open project folder
2. Double-click index.html
